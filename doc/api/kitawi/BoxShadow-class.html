BoxShadow

BoxShadow class

The BoxShadow class is used to add a box shadow to a widget It has a color Color, offset x, offset y, blur radius, and spread radius The offset x and offset y are the distance of the shadow from the widget The blur radius is the blur of the shadow The spread radius is the spread of the shadow The color is the color of the shadow
