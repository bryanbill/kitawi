Button

Button class

A button widget that can be clicked to trigger an action.

The Button widget allows you to create a button element with a given child widget and an action to perform when it's clicked. You can also specify various decorations like the background color, padding, margin, width and height of the button.

Example usage:

Button(
  child: Text('Click me'),
  action: Action(type: 'click', callback: (event) => print('Button clicked')),
  decoration: BoxDecoration(color: Colors.blue),
  padding: EdgeInsets.symmetric(horizontal: 16, vertical: 8),
  margin: EdgeInsets.only(top: 16),
  width: 200,
  height: 50,
)
