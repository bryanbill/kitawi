mode property

ThemeMode mode

Returns the current theme mode.

Implementation

ThemeMode get mode {
  if (_mode == ThemeMode.system) {
    return window.matchMedia('(prefers-color-scheme: dark)').matches
        ? ThemeMode.dark
        : ThemeMode.light;
  } else {
    return _mode;
  }
}
void mode=(ThemeMode value)

Implementation

set mode(ThemeMode value) {
  _mode = value;
